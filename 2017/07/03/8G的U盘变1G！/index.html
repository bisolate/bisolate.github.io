8G的U盘变1G！

Posted by Benjamin Benjamin on Jul 03, 2017

听说U盘会缩水，没想到缩这么厉害。我有个U盘，8G的，许久没用，一插上就说要先格式化才能用。格就格嘛，反正也没什么重要的东西，要命的是，格式化的界面，赫然显示容量只有1G！

当然这事没法找商家，因为一来是送的，二来时间也很久了，就算是买的也过保了，三来，我之前用过，并没有缩水这么严重啊。

好在不是我一个人遇到这个问题，所以一搜就搜到了，是因为我之前拿它做过启动盘。打开“磁盘管理”一看，一个盘变成三个盘：1G，12M，6G。因为中间有这个12M的东西，我把另俩删除也没用，它们合不到一块儿。只能借助 Diskpart 了。

  • 按 Win + R ，输入 cmd 调出“命令提示符”，再输入 diskpart，会弹出另一个窗口，即 diskpart.exe。
  • 输入 list disk，界面会列出所有的磁盘，序号从0开始。
  • 找到U盘对应的序号，比如我的是“2”，输入 select disk 2，回车。
  • 输入 clean，即可将磁盘完全清除。所以这一步要特别谨慎，千万要确定确实选的是U盘，别把自己的硬盘清空了。
  • “开始”菜单处点右键，找到“磁盘管理”，再找到U盘对应的区域，可以看到刚才的三个盘已经合并成一个盘。右键新建简单卷。Done！
